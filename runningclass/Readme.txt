Steps to add the class files to library

Prerequisite: Copy this folder to TYSelenium project

1. Open Eclipse and refresh the project to ensure the folder is copied successfully
2. Click on the Project tab
3. Click on Properties
4. Select Java Build Path from the left menu
5. Select Libraries tab
6. Click on Add Class Folder button
7. Select the folder runningclass and click OK
8. Verify that the folder is added to the list
9. Click on Apply and Close
10. Verify that the folder appears in your project as a library folder (icon will change)
